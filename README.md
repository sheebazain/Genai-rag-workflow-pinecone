# Enterprise AI Automation Suite

This project demonstrates enterprise-grade AI automation workflows using n8n, Pinecone, OpenAI, and Google Gemini.

## Features

- AI-powered customer support automation
- Retrieval-Augmented Generation (RAG)
- Semantic enterprise knowledge retrieval
- Gmail automation workflows
- Pinecone vector search
- AI-generated contextual responses
- Workflow orchestration using n8n

---

# Tech Stack

- n8n
- OpenAI
- Google Gemini API
- Pinecone Vector Database
- Gmail API
- REST APIs
- JSON

# Workflows Included

---

## 1. Enterprise RAG Knowledge Assistant

User queries are:
- converted into vector embeddings
- searched semantically in Pinecone
- matched against enterprise troubleshooting knowledge
- answered using LLM contextual generation

---

# Architecture

User Query → Embeddings → Pinecone Search → Context Retrieval → LLM Response

Incoming Email → AI Agent → Pinecone KB → AI Reply → Gmail Response
